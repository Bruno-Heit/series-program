 Este proyecto es hecho con la intención de crear una aplicación de escritorio que permita hacer un 
seguimiento de las series que el usuario ve, vió o tiene la intención de ver.
 NOTA: EL PROYECTO SE CENTRA EN SERIES E INCLUSO ANIMES, NO EN PELÍCULAS :( .
 Para eso el usuario debe poder:

    • (CREATE) Agregar nuevas series (junto con sus datos)
    • (READ) Ver los datos de las series
    • (UPDATE) Modificar los datos de las series
    • (DELETE) Eliminar series

 Implementación:
    El código está escrito principalmente en Python. De ser necesario, algunos módulos estarán escritos 
    en C -principalmente para una mayor velocidad de ejecución de las instrucciones-.
    Adicionalmente, decidí usar bases de datos SQL con SQLite para guardar la información, aunque pude 
    también usar archivos .csv, .txt, archivos excel, binarios, etc.
